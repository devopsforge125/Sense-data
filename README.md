## How to use this repo

1. Open **[INTERNSHIP_TASKS.md](./INTERNSHIP_TASKS.md)** and follow the steps in order.
2. Each section tells you exactly what to do to earn that badge.
3. Check off badges above as you earn them (or use the command below to mark one as earned).

### Command: mark a badge as earned

From the repo root, run:

```bash
# Interactive menu (choose by number or name)
python scripts/badge.py

# Or pass the badge name
python scripts/badge.py "Pull Shark"
python scripts/badge.py pull-shark
```

On Windows you can also run: `badge "Pull Shark"` or `badge` for the menu.

See **[CONTRIBUTING.md](./CONTRIBUTING.md)** for co-authored commits and how to open PRs.
