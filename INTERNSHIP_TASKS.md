# Internship Tasks — Earn Every Badge (Sense-data)

Complete these tasks in the **[Sense-data](https://github.com/superDev427/Sense-data)** repo. Each section earns one GitHub Achievement badge.

---

## Task 1: Pull Shark 🦈

**Badge:** Merge 2 pull requests

### Steps

1. **First PR**
   - Create a new branch: `git checkout -b task-1-add-bio`
   - Edit `participants/your-name.md` (or create it from `participants/example.md`).
   - Add your name and a short bio.
   - Commit, push, open a PR to `main`, then **merge** it.

2. **Second PR**
   - Create another branch: `git checkout -b task-2-fix-typo`
   - Fix a typo in `README.md` or add a line to `contributions.md`.
   - Commit, push, open a PR, then **merge** it.

✅ After merging **2 PRs** in Sense-data, you get **Pull Shark**.

---

## Task 2: Pair Extraordinaire 🤝

**Badge:** Co-author on 2+ merged pull requests

### Steps

1. **Co-authored commit**
   - In your next commit, add a **co-author** in the commit message:
   ```
   Add feature X

   Co-authored-by: Friend Name <friend@example.com>
   ```
   - Open a PR in Sense-data and **merge** it.

2. **Second co-authored PR**
   - Do one more PR with a `Co-authored-by:` line. Merge it.

✅ After **2 merged PRs** with co-authored commits, you get **Pair Extraordinaire**.

See [CONTRIBUTING.md](./CONTRIBUTING.md) for exact commit format.

---

## Task 3: Quickdraw ⚡

**Badge:** Close an issue or PR within 5 minutes of opening it

### Steps

1. On [Sense-data](https://github.com/superDev427/Sense-data), go to **Issues** → **New issue**.
2. Title: `Quickdraw practice`, body: `Closing this in under 5 minutes!`
3. Submit the issue.
4. **Within 5 minutes**, close the issue.

✅ You get **Quickdraw** when GitHub sees it closed within 5 minutes.

---

## Task 4: Starstruck 🌟

**Badge:** Create a repo that gets 16+ stars

### Steps

1. Keep Sense-data **public** and add a clear README (you’re already doing that).
2. Share the repo: Twitter/X, LinkedIn, Reddit, Discord, friends.
3. Ask people to star it if they find it useful.

✅ When [Sense-data](https://github.com/superDev427/Sense-data) hits **16 stars**, you get **Starstruck**.

---

## Task 5: Galaxy Brain 🧠

**Badge:** Get 2 accepted answers in Discussions

### Steps

1. In Sense-data: **Settings → General → Discussions** → enable **Discussions**.
2. In the **Discussions** tab, create a **Q&A**, e.g. "How do I add a co-author to a commit?"
3. Answer it (from another account or have a friend ask and you answer).
4. Mark your answer as **Accepted**.
5. Repeat with a second Q&A.

✅ After **2 accepted answers**, you get **Galaxy Brain**.

---

## Task 6: YOLO 🎲

**Badge:** Merge a pull request without a code review

### Steps

1. Create a branch, make a tiny change (e.g. add a line to `contributions.md`).
2. Open a PR to `main` in Sense-data. **Do not** request a review.
3. Merge the PR yourself.

✅ Merging a PR without review earns **YOLO**.

---

## Task 7: Open Sourcerer 🎖️

**Badge:** Have pull requests merged in 5+ different public repositories

### Steps

1. Sense-data counts as **1** (after you merge PRs here).
2. Find **4 more** repos (e.g. [good-first-issue](https://github.com/topics/good-first-issue)), open a small PR in each, get them merged.

✅ After PRs merged in **5 different repos**, you get **Open Sourcerer**.

---

## Task 8: Public Sponsor 💜

**Badge:** Sponsor open source via GitHub Sponsors

### Steps

1. Go to [github.com/sponsors](https://github.com/sponsors).
2. Pick a developer or org, click **Sponsor**, complete payment (even $1/month if allowed).

✅ After your first sponsorship, you get **Public Sponsor**.
