# Contributing to Sense-data

## Co-authored commits (Pair Extraordinaire)

Add this at the end of your commit message:

```
Your commit message here

Co-authored-by: Full Name <email@example.com>
```

Use the **email** tied to their GitHub account. One line per co-author.

### Example

```bash
git add .
git commit -m "Add new feature

Co-authored-by: Jane Doe <jane@example.com>"
git push origin your-branch
```

---

## Opening a pull request

1. Clone: `git clone https://github.com/superDev427/Sense-data.git`
2. Create a branch: `git checkout -b task-pull-shark-1`
3. Edit files (e.g. `participants/your-name.md`, `contributions.md`).
4. Commit (add `Co-authored-by:` if doing Pair Extraordinaire), then push.
5. On GitHub, open **Pull request** from your branch to `main`.
6. Merge (yourself or maintainer).

---

## Files you can edit for practice PRs

| File | Use for |
|------|--------|
| `participants/your-name.md` | Add yourself (copy from `participants/example.md`). |
| `contributions.md` | Add a line with your name and what you did. |
| `README.md` | Fix typos or improve wording. |

No need to ask—just open a PR. Good luck earning badges!
